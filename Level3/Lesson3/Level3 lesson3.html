Lesson 3: Loops in Python

1. Introduction to Loops

Loops are a fundamental concept in programming that allow you to repeat a block of code multiple times.

Loop Type Description When to Use
for loop Iterates over a sequence When you know how many times to repeat
while loop Repeats while condition is true When you don't know how many times to repeat

2. For Loop

Used for iterating over a sequence and when you want to repeat a block a fixed number of times.

# Basic for loop example
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Output:

apple
banana
cherry

Range Function

The range() function returns a sequence of numbers.

Syntax: range(start, stop, step)

for i in range(5):
    print(i)

Output: 0 1 2 3 4

for i in range(2, 6):
    print(i)

Output: 2 3 4 5

for i in range(0, 10, 2):
    print(i)

Output: 0 2 4 6 8

3. While Loop

Executes a set of statements as long as a condition is true.

# Password validation example
password = "python123"
user_input = ""

while user_input != password:
    user_input = input("Enter password: ")
    if user_input != password:
        print("Invalid password!")
        
print("Correct password!")
# Counter example
i = 1
while i < 6:
    print(i)
    i += 1

Output:

1
2
3
4
5
