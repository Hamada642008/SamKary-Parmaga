Lesson 2: Operators & Math Functions

1. Identity Operators

Compare if two variables refer to the same object in memory:

Operator Description Example
is Returns True if both variables are the same object x is y
is not Returns True if both variables are not the same object x is not y 
x = ["apple", "banana"]
y = x  # Same object
print(x is y)  # Output: True

2. Membership Operators

Check if a value exists in a sequence (like list or string):

Operator Description Example
in Returns True if value exists "a" in ["a", "b"]
not in Returns True if value doesn't exist "x" not in ["a", "b"] 
fruits = ["apple", "banana"]
print("apple" in fruits)  # Output: True

3. Math Functions

Basic Functions:

Function Description Example
min() Finds smallest value min([3, 1, 4]) → 1
max() Finds largest value max([3, 1, 4]) → 4
abs() Absolute value abs(-5) → 5
pow(x, y) x to the power of y pow(2, 3) → 8

Advanced Functions (requires import math):

import math

print(math.ceil(4.2))   # Output: 5 (round up)
print(math.floor(4.9))  # Output: 4 (round down)
print(math.sqrt(16))    # Output: 4.0 (square root)
print(math.pi)          # Output: 3.141592653589793
