List

Lists: used to store multiple items in a single variable using square brackets [].

Benefits of Lists:

  • Organize data
  • Store large amounts of data
  • Quick access to any item
  • Can store different data types (preferably same type)
  • Can loop through list items
  • Easy to add and remove items

How to create a list:

To create a list, use square brackets and assign it to a variable:

friends = ['yousef','abdo','anna','hesham','lujina']
print(friends)

Output:

['yousef', 'abdo', 'anna', 'hesham', 'lujina']

List Methods with Examples:

append(): Adds an element at the end of the list
friends.append('ahmed')
print(friends)

Output:

['yousef', 'abdo', 'anna', 'hesham', 'lujina', 'ahmed']
insert(): Adds an element at the specified position
friends.insert(1, 'sara')
print(friends)

Output:

['yousef', 'sara', 'abdo', 'anna', 'hesham', 'lujina', 'ahmed']
pop(): Removes the element at the specified position
friends.pop()
print(friends)

Output:

['yousef', 'sara', 'abdo', 'anna', 'hesham', 'lujina']
remove(): Removes the item with the specified value
friends.remove('abdo')
print(friends)

Output:

['yousef', 'sara', 'anna', 'hesham', 'lujina']
clear(): Removes all elements from the list
friends.clear()
print(friends)

Output:

[]
sort(): Sorts the list in ascending order
friends = ['yousef','abdo','anna','hesham','lujina']
friends.sort()
print(friends)

Output:

['abdo', 'anna', 'hesham', 'lujina', 'yousef']
reverse(): Reverses the order of the list
friends.reverse()
print(friends)

Output:

['yousef', 'lujina', 'hesham', 'anna', 'abdo']
index(): Returns the index of the specified element
i = friends.index('anna')
print(i)

Output:

3
extend(): Adds elements of another list
friends.extend(['omar', 'jana'])
print(friends)

Output:

['yousef', 'lujina', 'hesham', 'anna', 'abdo', 'omar', 'jana']
