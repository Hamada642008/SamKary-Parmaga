Lesson 4: Nested For Loops & Statements in Loops

1. Nested Loops

A nested loop is a loop inside another loop. Used when you need to iterate over multiple sequences.

Concept Description Example
Basic Nested Loop Loop inside another loop for x in seq1:
  for y in seq2:
Practical Use Combine all items from two lists for car in cars:
  for color in colors:		 
# Nested loop example
cars = ["BMW", "Toyota", "Honda"]
colors = ["red", "blue", "green"]

for car in cars:
    for color in colors:
        print(f"{car} is {color}")

Output:

BMW is red
BMW is blue
BMW is green
Toyota is red
Toyota is blue
Toyota is green
Honda is red
Honda is blue
Honda is green

2. Loop Control Statements

Special statements that change how loops execute:

Statement Description Example
break Stops the loop completely if x == 5: break
continue Skips current iteration if x == 5: continue
pass Placeholder for empty loops for x in seq: pass

break Statement

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    if fruit == "banana":
        break
    print(fruit)

Output: apple

continue Statement

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    if fruit == "banana":
        continue
    print(fruit)

Output: apple cherry

pass Statement

for x in [1, 2, 3]:
    pass  # Does nothing

Output: (No output)
