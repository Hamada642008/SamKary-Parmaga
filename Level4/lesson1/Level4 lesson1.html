Level 4

Lesson 1: Tuples

1. Introduction to Tuples

In the previous level, you learned about lists. Now you'll learn about tuples - an ordered and unchangeable collection.

Key Difference: Tuples are immutable (cannot be changed after creation).

Feature Tuple List
Syntax () []
Mutable No Yes
Use Case Data that shouldn't change (like national ID) Data that may need modification 
# Tuple creation
my_info = ("Mahmoud", "2005-01-01", "123456")
print(my_info[0])  # Access first element

Output:

Mahmoud

2. Modifying Tuples (Indirectly)

While tuples are immutable, you can modify them indirectly by converting to a list and back:

Step-by-Step Process

  1. Convert tuple to list
  2. Modify the list
  3. Convert back to tuple
# Original tuple
my_tuple = (1, 2, 3)

# Convert to list
temp_list = list(my_tuple)
temp_list[1] = 5  # Modify

# Convert back to tuple
my_tuple = tuple(temp_list)
print(my_tuple)

Output: (1, 5, 3)

3. Tuple Methods

Tuples have limited methods since they're immutable:

Method Description Example
count() Returns number of occurrences of a value my_tuple.count(2)
index() Returns index of first occurrence my_tuple.index(3)

count() Method

numbers = (1, 2, 3, 2, 4, 2)
print(numbers.count(2))

Output: 3

index() Method

numbers = (10, 20, 30, 40)
print(numbers.index(30))

Output: 2
