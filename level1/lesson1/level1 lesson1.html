Adding multiple messages using + or , :

Examples:

  • print("hello" + " python")
  • print("welcome", 25)

Output:

  • hello python
  • welcome 25

Examples:

  • print("welcome \n python")

  • print('''\n welcome\n python\n ''')

Output:

  • welcome
    python
