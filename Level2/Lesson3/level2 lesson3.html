Conditions

Conditions: Control the flow of code execution based on conditions and allow you to control the flow of your program based on whether certain conditions are true or false.

1. If: is used to execute a block of code, only if the condition specified holds true.

Examples:

  • x=4
  • y=5
  • if x<y 
  •   print('yes')

Output:

  • yes
2. Elif: is used for multiple condition checking in an if statement and like the if syntax .

Examples:

  • x=4
  • y=5
  • if x>y:
  •   print('False x>y')
  • elif x<y:
  •   print('true x<y')
  • else:
  •   print('error')

Output:

  • true x<y

3. Else: is used to execute a block of code if the same condition is false.

Examples:

  • x=4
  • y=5
  • if x>y:
  •   print(False)
  • else:
  •   print('error')

Output:

  • error
